# Lab 9 - Exercise 3
## React Counter Application

### Aim
To develop a simple counter application using React useState Hook.

### Description
This application displays a counter value and provides Increment and Decrement buttons. The counter value updates dynamically using React state and event handling.

### Technologies Used
- React JS
- JavaScript
- JSX
- useState Hook

### Result
The counter value increases and decreases when the user clicks the buttons.
